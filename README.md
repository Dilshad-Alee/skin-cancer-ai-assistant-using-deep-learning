# 🩺 Skin Cancer AI Assistant

> **Disclaimer:** This tool is for informational and educational purposes only. It is **not** a medical diagnosis. Always consult a licensed dermatologist or healthcare professional.

A full-stack AI-powered web application that classifies skin lesion images as **Benign** or **Malignant** using a custom-trained Convolutional Neural Network, with natural-language explanations generated by Google Gemini.

---

## 🚀 Features

- 🔬 **Binary Classification** — Benign vs Malignant using a trained CNN (`.keras`)
- 📊 **Confidence Score** — Visual confidence bar per class
- 🧠 **AI Explanation** — Google Gemini generates a plain-language, medically-safe explanation
- 🗺️ **Grad-CAM Heatmap** — Highlights which regions of the image influenced the model's decision
- 📄 **PDF Report** — Download a complete prediction report per analysis
- 🕓 **Prediction History** — Locally saved history of past predictions
- 🐳 **Dockerized** — One command to build and run the entire backend
- 📱 **Mobile Responsive** — Clean medical-themed UI works on all screen sizes

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite + Tailwind CSS |
| Backend | FastAPI + Uvicorn (Python 3.11) |
| ML Model | TensorFlow / Keras CNN |
| Explainability | Grad-CAM via TensorFlow + OpenCV |
| Generative AI | Google Gemini API (`gemini-2.0-flash`) |
| PDF Export | jsPDF (client-side) |
| Containerization | Docker + Docker Compose |
